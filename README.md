# Portfolio – John Doe

Ce projet est un portfolio de John Doe, un futur développeur web full stack, développé avec Next.js, React et Bootstrap, dans le cadre du devoir final de la partie Front-End de la formation développeur web et web mobile du Centre Européen de Formation.  
Il présente ses compétences, ses services et ses réalisations.


# Prérequis

- Node.js 
- npm
- Navigateur internet (Chrome, Firefox, Edge, Safari)


# Installation

1. Cloner le dépôt :

git clone https://github.com/BrochotMaxime/brochot-maxime-devoir-portfolio

2. Se déplacer dans le dossier du projet :

cd portfolio

3. Installer les dépendances :

npm install


# Lancement du projet 

1. Lancer le serveur de développement :

npm run dev

2. Ouvrir le navigateur internet à l'adresse suivante :

http://localhost:3000

Le portfolio est alors accessible pour consultation.
